MacroMonitor

MacroMonitor is a user-friendly app designed to help individuals track their daily intake of macronutrients—proteins, fats, and carbs. Set goals, monitor progress, and optimize your diet for better health results.

The application was created using Flutter and Dart. It includes several libraries, most notably Google’s Machine Learning Kit.
